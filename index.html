    • Summary: Lattitude vs. Weather Condition

    This is an analysis and visualization of weather of 400+ cities across the world. The purpose of this analysis is to visualize the effect of lattiude on different weather conditions, including cloudiness, temperature, wind speed, and humidity
    We first randomized created few thousands coordinates pairs and then used citypy to find the nearst cities to these coordinates pairs. After collecting these cities coordinates, we then used openweather api to find out the weather conditions of these cities.
    In conclusion, an we found out that except for temperature, there are not significant effects on cloudiness, windspeed, nor humidity with lattitude.
